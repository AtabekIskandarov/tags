Форматирование текста

h1-h6 - Tеги заголовков, от самого большого до самого малеьнкого.
b - Жирный текст - без придания важности выделенному фрагменту
i - Выделение текста курсивом без придания важности
del - Зачеркивает текст, помечая его удаленным
s - Отображает перечёркнутй текст
ins - Подчёркивает текст, визуально выделяя внесённые изменения
u - Подчёркивание без дополнительного акцентирования внимания
em - Pасставление акцентов путём выделения фрагментов текста курсивом
mark - выделения частей текста жёлтым маркером
tt - имитация текста, набранного на печатной машинке
small - отображение фрагмента с меньшим кеглем шрифта, чем у остального текста
sub - подстрочное начертание символов
sup - надстрочное начертание символов
cite - оформление цитат
address - добавление контактов или подписей автора. При открытии веб-браузере выделяется курсивом
pre - вывод неформтированного текста с сохранением пробелов и особенностей переносов
p - контейнер для абзаца
br - переносит текст на другую строку без создания абзаца
blockquote - отступы с обеих сторон для оформления цитаты или вырезки
q - краткое цитирование
dl - контейнер для размещения термина и его определения
dt - добавление термина
dd - добавление опредения понятия
dfn - выделение термина курсивом. Последующий текст должен расскрывать понятие
abbr - указывает, что текст является аббревиатурой или акронимом. Для добавлния пояснения используется атрибут title
ol - список с цифрами
ul - список со значками
li - отметка каждого элемента перечня (цифра или значок в зависимости от типа списка)
a - добавление гипперсылки в текст. Имеет обязательный атрибут
code - выделение фрагмента кода с помощью шрифта monospace

Встраивание элементов

img - вставка изображения. Имеет атрибут src который указывает адрес нужного файла. Есть и другие атрибуты.

img src="name" align="?" - выравнивание к одной из сторон документа. Например, значение right переместить рисунок в правый край, а left - в левый.

img src="name" border="?" - позволяет настроить в пикселях толщину рамки вокруг изображения.

picture - контейнер для расположения элемента img без source, который даёт браузеру возможность самостоятельно выбрать подходящую картину.

audio - вставка звукового контента

video> - вставка видео (поддерживается Ogg, WebM, Mp4)

source - указывает местоположение файла для video, audio и picture

track - формирует субтитры для video и audio

object - контейнер, через который втраевается мультимедийные файлы. Для передачи параметры применяется тег param

hr - размещает на странице горизантальную линию. Имеет несколько атрибутов.

hr size - устанавливает высоту линии

hr wigth - устанавливает ширину линии

hr noshade - убирает тень у линии

hr color - задает цвет линии

script - определяет выполнения сценария на стороне посетителя сайта. может содержать сам срикпт. или иметь ссылку на внешний источник.

noscript - ограничивает фрагмент документа, в котором скрипт не выполняется

Работа с таблицами

table - размешает таблицу. Все остальные теги для форматирования таблиц должны назодиться внутри этого контейнера.

thead - определяет заголовок.

tbody - отмечает тело таблтцы

td - создаёт одну ячейку

tr - создание одной строки

tfoot - показывает нижний колонтур

caption - вставляет подпись Указывает после тега table

col - позваляет указывать ширину и другие параметры одной или нескольких колонок

Создание форм и кнопок

form - создание форм на странице. С помощью HTML описывается только внешний вид, для выполнения функций требуется запуск соответствующих скриптов на сервере

button - создаёт интерактивную кнопку. Внутри контейнера нужно поместить текст или изображение.

select multiple name="NAME" size="?" - формирует меню с поддержкой скроллинга.

option - описывает каждый отдельный пункт меню.

input - формирует поля для добавления пользовательских данных.

output - выводит результаты вычислений, сделанных с помощью скрипта.

label - оформляет пометку для поля, созданного тегом input

textarea - создает большие поля для ввода текста.
